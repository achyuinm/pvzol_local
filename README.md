## 简介

这是一个以**客户端真实行为**为准、以**抓包与源码证据**为依据的项目。

它不是示例。  
不是模板。  
也不是“可以随便优化结构”的那一类仓库。

这是一个**被观察、被验证、被复现**出来的系统。

---

## 项目原则（先读）

- **证据优先于猜测**
- **结构优先于感觉**
- **能跑不等于正确**
- **能改不等于应该改**

如果你不同意这些原则，你仍然可以阅读。  
但请不要提交修改建议。

## 目录结构（权威定义）

```text
cache/   ← 已分发的客户端静态资源（只读）
